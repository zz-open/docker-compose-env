- data 4个节点
- sql 2个节点
- phpmyadmin


## 部署的注意事项
- windows下各种问题，请自行修改
- 容器名称禁止修改，配置文件中也用了
- nodeid禁止修改
- mysql密码禁止修改，因为sql文件中也用到了
- 建表的Engine=NDBCLUSTER, 禁止修改
- 大约30秒之后才可以启动完成
- 推荐使用自定义用户连接mysql操作
